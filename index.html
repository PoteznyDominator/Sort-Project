About Site

A Sorting Algorithm is used to rearrange a given array or list elements according to a comparison operator on the elements. The comparison operator is used to decide the new order of element in the respective data structure. My site visually represent different types of algorithms and how fast each one is. Every sorting has time and space (memory) complexity. You can check more about each alghoritm by clicking info icon.
